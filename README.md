# Configurações #

## Pré-Requisitos: ##
- Java 11
- Maven 3.9.12

### Execução do projeto e geração de relatórios ###
Para geração de relatórios e algumas configurações básicas, está sendo utilizando a biblioteca do SerenityBDD.

Para executar o projeto, basta acessar a pasta do projeto e executar o comando 'mvn clen verify'.
Os testes serão executados e o relatório será gerado no diretório do projeto, na diretório 'target/site/serenity/index.html'
